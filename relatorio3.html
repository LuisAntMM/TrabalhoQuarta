Resumo do projeto

Este relatório descreve a implementação de um sistema de indicadores visuais usando um Arduino Uno, um sensor ultrasônico e um LED RGB. O objetivo principal foi demonstrar a integração desses componentes para criar um sistema que forneça feedback visual com base nas medições de distância obtidas pelo sensor. O LED RGB acende em diferentes cores para indicar a proximidade de um objeto detectado pelo sensor ultrasônico, com base em intervalos de distância específicos. O projeto envolveu a configuração do sensor ultrasônico para medir a distância e a programação do Arduino para controlar o LED RGB, garantindo que o sistema funcionasse conforme o esperado.
